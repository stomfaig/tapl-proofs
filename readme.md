# TAPL proofs in LEAN4

This repo is a collection of formal proofs to the theorems in the book Types and Programming Languages. 

Currently proofs are availble for:
- All major results including boolean terms from Chapter 3 (including defintions, inudtion principles, normal forms, reducibility to normal form etc.)
- The proof of safety and progress theorems from Chapter 9 for the simply typed lambda calculus with Booleans. The formalisation uses nameless terms, and therefore deviates from the proofs presented in the book with several auxiliary technical lemmas.

The long term goal of this project is to formalize as much of TAPL and Advanced Topics in TAPL as possible, and provide a good starting point for formal verification of results in the PL domain.
